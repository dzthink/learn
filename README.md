# AI 内容创作工作流模板仓库

这是一个为「AI 驱动的内容创作工作流」设计的 GitHub 仓库模板，用于承载：

- 从 Draft → Idea → Article → Derivatives 的内容演化
- 所有文章、文集、视频/播客脚本等内容资产
- 未来自动发布系统（FastAPI / GitHub Actions 等）的基础

## 目录结构概览

- `content/`：所有内容资产（草稿 / 大纲 / 文章 / 衍生内容 / 资源）
- `docs/`：工作流说明文档等
- `metadata/`：索引和状态数据（可选）

你可以直接将此仓库推送到 GitHub，并在此基础上扩展自动化发布系统。
